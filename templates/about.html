About This Document Scanner App

Welcome to my Document Scanner App! This app was designed to provide you with a document scanning experience similar to CamScanner but without the annoying watermarks.

Features

  • No Watermarks: Enjoy clean and professional scans without any watermarks.
  • High-Quality Scans: Leverages OpenCV for superior image processing.
  • User-Friendly Interface: Easy to use with a simple and intuitive interface.
  • Auto-Edge Detection: Automatically detects and crops document edges for perfect scans.

Technologies Used

This Document Scanner App was built using the following technologies:

  • OpenCV: For advanced image processing and computer vision tasks.
  • Python: The core programming language used for the backend logic.
  • Flask: The lightweight web framework used for building the backend.
  • JavaScript: For creating a responsive and dynamic user interface.
  • Bootstrap: To ensure a responsive design that works across all devices.
  • HTML & CSS: For structuring and styling the web pages.
